BESTMAP Habitat Suitability and Biodiversity Indicator

The main objective of the model is the estimation of how the adoption of Agri-Environment Schemes affectshabitat suitability for a set of selected farmland bird species according to different scenarios and case studies. The output results of this model are the following layers:

  • Habitat suitability: The habitat suitability layer has values that range from 0 (unsuitable habitat) to 1000 (highly suitable habitat). (The uncertainty maps are calculated as standard deviation across the single models algorithms used to compute the ensemble models). The user can click on the down-arrow of "Species" to display the list of bird species from which they can select one single bird species. Next, the user can select one of 4 different scenarios from the dropdown list "Scenario". The 4 possible scenarios are:

    • Current: scenario with current Agri-Environment Schemes (AES) adoption;
    • Current SD: uncertainty estimates calculated as the standard deviation in a scenario with current Agri-Environment Schemes (AES) adoption;
    • No AES: scenario without Agri-Environment Schemes (AES) adoption;
    • No AES SD: uncertainty estimates calculated as the standard deviationin a scenario without Agri-Environment Schemes (AES) adoption.

  • BESTMAP Biodiversity indicator: this indicator consists of a relative species richness index that ranges from 0 to 1. The relative species richness index is derived from the habitat suitability maps of individual farmland bird species. A relative species richness index of 0 indicates unsuitable habitat for all modelled species, whereas 1 indicates that the habitat is suitable for all modelled species. The user can select 2 different scenarios from the dropdown list "Scenario". The 2 possible scenarios are:

    • Current: scenario with current Agri-Environment Schemes (AES) adoption;
    • No AES: scenario without Agri-Environment Schemes (AES) adoption.
